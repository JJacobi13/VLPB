convertToBam(sample)[source]¶

The method convertToBam converts a sam file to a bam file of a given sample, if the file is not a sam file, the file first will be mapped by calling the Mapper.Mapper.map() function.

Parameters:sample (an instance of a Sample object) – The sample which contains the sam file to be converted to a bam file

removeDuplicates(sample)[source]¶

The method removeDuplicates removes all possible PCR duplicates of the bam file of a given sample. If the input file is not a sorted bam file, first the method sortBam() will be executed.

Parameters:sample (an instance of a Sample object) – The sample which contains the bam file to remove the duplicates from
sortBam(sample)[source]¶

The method sortBam sorts a bam file of a given sample. The bam file can only be sorted when this is a bam file. If it is not a bam file, the file first will be converted to a bam file.

Parameters:sample (an instance of a Sample object) – The sample which contains the bam file to be sorted

callHaplotypes()[source]¶

The method callHaplotypes calls the haplotypes of a vcf file. When the vcf file is not set, first the SNVs are called with the SnvCaller.SnvCaller object In future when multiple programs for haplotyping are implemented, here the check will be performed which haplotyper to execute. First only beagle is implemented so this method will be called :param pool: The pool a haplotyping tools has to be executed on :type pool: an instance of a Pool.Pool object
