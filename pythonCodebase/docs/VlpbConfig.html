VlpbConfig Package¶

Configuration Module¶

The configuration module regulates all configuration settings which are in the ini file. This is done by 2 objects, Config and Section. The Config regulates the parsing of the ini file and the communication with other objects The Section represents a section of the ini file

class VlpbConfig.Configuration.Config[source]¶

Bases: builtins.object

The Config class regulates everything of the configuration of the programs. This information is read from the iniFile config.ini which is located in this directory.

The class section represents a section from the ini file. This section contains a header name and a dictionary of key-value pairs. The value is an empty string if this option has no value

logError(errorMessage)[source]¶

The method logError writes errors to the log file A little ascii art is also created so the user can scan easy for errors. This error is only printed if the message is not empty!

Parameters:errorMessage (str.) – the error message which has to be printed.
logOut(outLines)[source]¶

The method logOut writes the output of the programs to the log file.

Parameters:outLines (str.) – the lines which have to be written to the output file.

The method nextProgram print the next program with its command to the log file.

Parameters:

  • command (str.) – The command which is executed right now
