Bases: File.File

The BeagleFile represents a BeagleFile file and contains instance variables to indicate the processes used on this BeagleFile file. The beaglefile contains multiple output files.

getFile(outFile=None)¶

The method getFile overrides the super method getFile. When there is no outFile given, the index is returned. Else the output file of the given outFile is returned. When an file is compressed, the decompressed file is returned! Beagle returns compressed files by default.

Parameters:outFile (“dose”, “gprobs” or “phased”) – The file where to get the (extracted) filename of
Returns:the name of the (extracted) file
